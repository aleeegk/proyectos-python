\# 🚀 Mini-Proyectos en Python: Scripts y Utilidades Esenciales



¡Bienvenido a mi colección de mini-proyectos de automatización y lógica en Python! Este repositorio contiene una serie de herramientas y utilidades interactivas desarrolladas en la terminal, ideales para practicar estructuras de datos (`lists`, `sets`, `tuples`), bucles (`while`, `for`) y manipulación de strings.



\---



\## 🛠️ Listado de Proyectos



\### 1. 📊 Analizadores y Contadores de Texto

\* \*\*Analizador de Frases:\*\* Evalúa dinámicamente un texto ingresado calculando de forma precisa el número de palabras, espacios, vocales, letras del abecedario y caracteres especiales.

\* \*\*Contador de Caracteres:\*\* Una versión simplificada centrada en desglosar las métricas de longitud del texto, espacios vacíos y vocales totales.

\* \*\*Conversor de Nombre:\*\* Utilidad de formato de strings que transforma cualquier nombre en mayúsculas, minúsculas, formato capitalizado (`Capitalize`) e invierte el orden del texto (lectura al revés).

\* \*\*Detector de Palíndromos:\*\* Comprueba de forma instantánea mediante técnicas de slicing si una palabra se lee igual de izquierda a derecha que de derecha a izquierda.



\### 2. 🧮 Utilidades Operativas y Seguridad

\* \*\*Calculadora Interactiva:\*\* Menú interactivo en bucle continuo que permite ejecutar operaciones matemáticas básicas y avanzadas:

&#x20;   \* Suma, Resta, Multiplicación y División.

&#x20;   \* División Exacta (entera), Potenciación y Módulo (resto).

&#x20;   \* Soporta números decimales (`float`) y detecta cuándo los resultados pueden simplificarse a enteros (`int`).

\* \*\*Generador de Contraseñas Básico:\*\* Algoritmo simple de encriptación que toma una palabra clave y genera un hash alfanumérico sustituyendo cada letra por su posición numérica correspondiente en el alfabeto.



\### 3. 📋 Gestores de Datos y Colecciones

\* \*\*Lista de Compra:\*\* Aplicación CRUD (Crear, Leer, Eliminar) en consola para gestionar un inventario de compras mediante el uso dinámico de listas (`.append()` y `.remove()`).

\* \*\*Gestor de Países Visitados:\*\* Bitácora de viajes avanzada que implementa un `set()` para evitar duplicados en los países, y listas paralelas de tuplas para almacenar de forma indexada el año del viaje y notas personalizadas. Permite:

&#x20;   \* Añadir países validando que solo contengan texto y años numéricos.

&#x20;   \* Visualizar el historial cronológico con sus respectivos comentarios.

&#x20;   \* Editar, reescribir o eliminar notas específicas de cada destino.

\* \*\*Tarjeta de Presentación (Básica y Mejorada):\*\* Interfaz de captura de datos de usuario que recopila información personal (nombre, edad, ciudad) y la imprime formateada y sanitizada usando concatenaciones y capitalización automatizada.
